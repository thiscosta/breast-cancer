
# Breast Cancer

A Python KNN implementation that determines cancer types (Benign/Malign), based on csv dataset


## Run Locally

Clone the project

```bash
  git clone https://github.com/thiscosta/breast-cancer.git
```

Go to the project directory

```bash
  cd breast-cancer
```

Install dependencies

```bash
  pip install numpy
  pip install pandas
```

Run the application

```bash
  python main.py
```
